Lets look at cycleGan first

Basic idea is you have image of one style ,you copy that style to another(Yeah like the Prisma APP which was so viral and you dowanloaded it then and you forgot to delete it. Now its just wasting space in your mobile device).

Cool thing is you only need unpaired image samples. So google some stuff copy some images and you have a dataset.

If you wanna know more read the paper. In short it has a cyclic loss with the Adversarial loss.

Our Tweak

Lets convert audio to image and apply the same thing.

So i did it on greyscale. Yes im too impatient to wait for an RGB to train to see if a hypothesis works.

So we took 20 second audio and converted to Greysacle image and applied CycleGAN.
